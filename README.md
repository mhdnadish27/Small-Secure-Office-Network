# Small Secure Office Network

This Cisco Packet Tracer project simulates a small secure office network using static IP addressing, VLANs, router-on-a-stick inter-VLAN routing, and basic network security configurations.

## Features
- Static IP configuration
- VLAN segmentation
- Inter-VLAN routing
- Router subinterfaces
- SSH configuration
- Port security
- ACL (Access Control List)
- Router password protection
- Basic firewall/security concepts
- Connectivity testing

## Devices Used
- Cisco 2911 Router
- Cisco 2960 Switch
- PCs
- Server

## Security Configurations
- Console password
- Enable secret password
- SSH remote access
- Port security with sticky MAC
- ACL traffic filtering

## Files Included
- Packet Tracer topology (.pkt)
- Router configuration screenshots
- Ping testing screenshots

- Cisco Packet Tracer
